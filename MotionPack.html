MO-CAP FOR UNITY 3D

Our dream has always been to create outstanding animations in very short times. This is key in many situations, especially when fast prototyping is needed to assess if an idea will be worth further development. Finally technology has caught up with this dream and we are able to provide others the option to easily create custom animation trees matching their needs. We are proud to present our first creation: expandable character controller which would allow indie developers to make games with high quality animations.

Thanks to Mechanim it is now possible to use mo-cap data to drive the characters in your game. It also gives you extensive control over the transitions between different motions and how they are aligned with each other. With such great tool at hand we were able to create a basic locomotion package that can be applied to any bipedal character with just few clicks and when needed, to add more animations or easily remove the unneeded ones.
